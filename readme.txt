Raft部分代码测试
修改时间：2020/6/10
测试人：Zavier Jin

文件结构说明：
	1. exe文件夹中存放可执行文件（包含三种测试样例）
    2. include文件夹中存放模块的头文件
    3. src文件夹存放各个模块的源代码
    4. RaftTest文件夹中存放测试工程，RaftTest.vcxproj可用vs打开
    5. lib文件夹中存放相关库
	
测试方法：
	1. 打开exe文件夹，选择需要测试的样例
	2. 首先运行clear.bat，会自动清空C:\network文件夹下的内容。
	3. 接着运行start.bat，会自动开启7个进程。（可能有点慢）
	4. 测试完成后，可运行kill.bat自动关闭7个进程。
	5. 每次测试完成，文件夹目录下会产生7个储存节点输出的文件（以对应节点的ID命名），可直接使用记事本打开进行查看和比对。
