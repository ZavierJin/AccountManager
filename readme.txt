Raft部分代码测试
修改时间：2020/6/10
测试人：金展羽
文件结构说明：
    1、include文件夹中存放模块的头文件
    2、src文件夹存放各个模块的源代码
    3、RaftTest文件夹中存放测试工程，RaftTest.vcxproj可用vs打开
    4、lib文件夹中存放相关库
	5、Debug文件夹中存放可执行文件和测试日志
	
测试方法：
	1. 清空C盘network文件夹下的内容！！！
	2. 打开RaftTest.exe进程，同时开四个（目前设置节点数为4，如需修改可在raft_type.h中NODE_TOTAL的值。
	3. 四个都开好后，正常测试开始，每个进程会有What...提示
	4. 最后需要手动关闭所有进程。每个节点的输出日志会在RaftTest.exe同目录文件夹下，以进程号命名。可以直接用记事本打开这些日志，进行查看。
